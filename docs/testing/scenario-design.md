# Scenario Design Methods

- Equivalence classes: valid/invalid emails, phone formats, required fields.
- Boundary values: empty strings, max length, 0/1 values, future dates.
- Decision tables: role x route access, invoice status x CTA.
- Pairwise: device x browser x role.
- Error injection: 401/403 RLS errors, 429 rate limit, 500 lead submit.
